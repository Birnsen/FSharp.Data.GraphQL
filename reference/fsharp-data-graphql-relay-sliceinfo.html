FSharp.Data.GraphQL

SliceInfo

Namespace: FSharp.Data.GraphQL.Relay

Slice info union describing Relay cursor progression.

Union Cases

Union CaseDescription
Backward(last,before)
Signature: int * string option

Return page of last results before provided cursor value. If before value was not provided, return last results of the result set.
Forward(first,after)
Signature: int * string option

Return page of first results after provided cursor value. If after value was not provided, start from the beginning of the result set.
